# Data type in JavaScript

> Data types 

```javascript
// string
var x = 'Hello world'
// array/object
x = [' html', 'css', 'js']
// object
x = { first: 'uzair', last: 'ahmed' }
// number
x = 69.9
// boolean
x = true
x = false
// null
x = null
// undefined
x = undefined
document.write(x)
document.write('<br>')
document.write(typeof x)
```
