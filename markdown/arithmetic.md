# Arithmetic Opration in JavaScript

> This is arithmetic oprations in JavaScript

```javascript
var a, b;
a = 10;
b = 20;
// multiplication
a * b;
// subtraction
a - b;
//addition
a + b;
// division
a / b;
// modulus(remainder)
a % b;
// exponentitaion
a ** b;
// increment
a ++ b;
// decrement
a -- b;
```
